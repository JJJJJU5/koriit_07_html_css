## HTML + CSS의 결과물 = '문서'

저희가 Java를 배우면서 작성한 것은 'code'라는 표현을 사용했지만 , HTML + CSS로 작업한 것은 '문서'라고 부름
HTML , CSS는 프로그래밍을 하는 것이 아니라는 의미

웹페이지에 들어갈 요소들을 배치하고 디자인하는 정도인 . html 파일과 .css파일은 브라우저라는 프로그램에서 불러 와 보여주는 문서(document)라고
생각하는 것

다만 JS역시 서버로부터 전달 받아 브라우저에서 동작하는 파일이지만 큰 차이점은 Node.js등 컴퓨터에 설치된
JS엔진 에서 python과 같은 언어처럼 복잡한 알고리즘을 정의할 수 있고 컴퓨터에 명령을 내릴 수 있기 때문에
Js는 프로그래밍 언어,Js로 작성한 것은 Code이지만 HTML,CSS의 결과물은 문서(document)라고 표현함

### 웹 퍼블리셔 vs 웹 개발자

웹 디자이너가 작업한 결과물을 HTML + CSS 문서로 작성하는 사람들을 우리 나라에서는 웹 퍼블리셔라고 부름
여기에 JS까지 적용해서 기능을 만들어 낼 수 있는 사람을 웹 개발자가 될 준비를 한 것

ch00_starting 폴더에 index.html을 생성했을때 가장 먼저 한 것은 `!+tab`이었음

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body></body>

```

`!+tab`으로 자동 생섣된 것은 이상과 같음, 이는 웹 개발 시에 기계 혹은 시스템에 전달하는 각종 정보로, **HTML문서 작성 시에 꼭 필요한 내용들을 담고 있음.**
문서 제목 , 작성자, 사용 언어 및 각종 메타 정보 등으로 구성되는데, 형식 자체는 동일하기 때문에 매번 부등호부터 시작해서 일일이 작성 할 필요가 없도록
단축키 지원을 함
Emmet이라고 하는데,`!+tab`도 이 기능 중 일부에 해당함.
앞으로 에밋이라고 했을 때 , 특정 문서각성 방식을 빠르게 해주는 단축키 정도로 인지하면 됨.
특히 `!+tab`은 html 파일을 생성할 때 마다 사용할 예정이기 떄문에 꼭 기억해둘 것

## 태그 사용 방식

```html

```

- 참고 : md에서는 emmet이 적용되지 않기 때문에 직접 여는 태그와 닫는 태그를 작성해야 함. -`<ㄱㄴㄷ>` : 여는 태그 -`</ㄱㄴㄷ>` : 닫는 태그
  여는 태그와 닫는 태그는 같은 코드 라인에 있을 수도 있고 여러 줄일 수도 있음
  여러 줄인 경우에는 여는 태그 와 닫는 태그 사이의 영역에 들여쓰기를 해주는 것이 가독성이 좋음
  code에서 자동 지원하기도 함.

```html
<body>
  자동으로 들여쓰기가 됨 하지만 브라우저 상에서는 전혀 티가 나지 않음, 단지
  사람이 알아보기 편하게 예의상 들여쓰기를 사용
</body>
```

에를 들어서 한국어 / 영어 매칭을 하는 1:1 매칭과 n:n 다대다 매칭을 하는 방식의 리스트 관리를 해야할 댸 사용하는 태그(잘 사용하지 않음)

형식

```html
<dl>
  <dt>용어</dt>
  <dd>정의</dd>
</dl>
```

와 같은 방식으로 작성됨
`<dl>` 태그 : definition list의 축약어로 정의 목록
`<dt>` 태그 : definition term의 축약어로 정의되어야 하는 용어
`<dd>` 태그 : definition description의 축약어로 용어의 설명 혹은 묘사

에시

```html
<body>
  <dl>
    <dt>프로그래밍</dt>
    <dd>컴퓨터 프로그램을 작성하는 일</dd>
    <dt>넓이</dt>
    <dt>광</dt>
    <dt>면적</dt>
    <dd>일정한 평면에 걸쳐 있는 공간이나 범위의 크기</dd>
    <dt>사과</dt>
    <dd>사과나무의 열매</dd>
    <dd>자기의 잘못을 인정하고 용서를 비는행위</dd>
  </dl>
</body>
```

4. `<td>` 태그 : table data cell의 축약어로 \_세로줄
